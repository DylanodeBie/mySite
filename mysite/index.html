Welkom!

Welkom om de informatie dag van Software Development Curio in Breda, Waarschijnlijk ben je geintresseerd in de opleiding software developement want ja anders zou je hier niet zitten natuurlijk :3 Deze opleiding wordt uitgeoefend in Breda op de locatie Terheijdenseweg 350 Op deze site kan je informatie vinden over de vakken die je krijgt, het verloop van de opleiding en het rooster. Deze opleiding is een 4 jarige opleiding waarbij je leert om software te maken. Als je nog vragen hebt kan je die stellen aan de docenten die aanwezig zijn op de opendag. Veel plezier op de opendag!
